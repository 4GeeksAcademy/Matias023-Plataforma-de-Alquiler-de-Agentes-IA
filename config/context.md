# AgentHub - Contexto del Producto

## Nombre del producto
AgentHub

## Descripcion breve
AgentHub es una plataforma SaaS donde empresas pueden alquilar agentes de IA preconfigurados para diferentes necesidades operativas.

## Usuario principal
Administrador interno del panel.

## Problema que resuelve
Centraliza la operacion administrativa para revisar clientes, usuarios, agentes, skills, contratos, errores y metricas operativas en un solo lugar.

## Alcance del prototipo
Panel de administracion estatico, sin backend, con datos hardcodeados.

## Secciones requeridas
- Dashboard
- Gestion de usuarios
- Gestion de agentes
- Skills
- Contrataciones de agentes
- Log de errores

## Restricciones tecnicas
- HTML
- Tailwind CSS via CDN
- JavaScript vanilla
- Sin frameworks
- Sin backend
- Sin archivos CSS externos
- Sin estilos inline
- Datos hardcodeados
